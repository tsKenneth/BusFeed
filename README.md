# BusFeed

## Data Sets
Bus arrival data and busstop data, obtained from
https://www.mytransport.sg/content/mytransport/home/dataMall/dynamic-data.html

## Methodology
Utilised PHP Guzzler to request data using the API and presented it in a responsive web application that imitates the UI of an Android mobile application.
